    Installation

    Dependencies

    • Boost
    • soci(sqlite3 backend)
    • yaml-cpp
    • cmake

    Building

    1. git clone git://github.com/theunknownxy/slin.git
    2. cd slin
    3. mkdir build && cd build && cmake .. && make && sudo make install
